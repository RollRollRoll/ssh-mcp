# SSH MCP 可视化工作台

这是 SSH MCP 的独立前端页面，用于展示主机拓扑、操作队列、快速命令和一次性审批流程。

当前版本使用演示数据，仅提供界面与交互原型；它不会直接连接 SSH 主机，也不会读取 SSH MCP 的配置、凭据或运行状态。将页面接入真实 MCP 客户端需要单独设计受信任的桥接层，不能从浏览器直接连接现有 `stdio` 服务。

## 本地开发

要求 Node.js `>=22.13.0`。

```bash
npm ci
npm run dev --workspace=web
```

Vite 默认访问地址为 `http://localhost:5173/`。

## 验证

```bash
npm run check
```

该命令会完成根项目与控制台的构建、类型检查、既有后端单元/契约/验收测试，以及控制台组件和静态构建产物测试。当前控制台仍是演示原型；服务端渲染、移动端兼容性和无障碍专项验证将在后续任务中补充。
